# Остров игр

Витрина мини‑игр для публикации на GitHub Pages.

## Как задеплоить на GitHub Pages
1. Откройте настройки репозитория на GitHub.
2. Перейдите в раздел `Pages`.
3. В `Source` выберите ветку `main` и папку `/ (root)`.
4. Сохраните настройки и дождитесь генерации.

Главная страница витрины находится в `index.html` в корне репозитория.
